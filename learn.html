Learn HumanNet

Understand the architecture and flow of the decentralized safety network.

HumanNet Concept

Community-powered emergency response: When danger strikes, HumanNet instantly notifies nearby volunteers. First 3 responders become Guiders who coordinate via VoIP/chat. Others join as support. Data auto-deletes after resolution. Focus on golden hour (first 60 min).

Network Actors

📱 Sender

Person in danger triggers SOS. Shares name, GPS, type. Triggers network broadcast.

🦸 Receiver (Volunteer)

Nearby users receive alerts. First 3: Guiders (lead response). Others assist.

👥 Crowd Reporter

Bystanders report public issues (fire, flood). Broadcasts to all nearby.

System Flow

Flow: SOS → Network Broadcast → Nearby Volunteers Alerted → Guiders Coordinate (VoIP) → Physical Response → SAFE → Data Deleted.
